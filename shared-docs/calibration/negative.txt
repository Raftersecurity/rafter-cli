# Seeds:
#   - rc-vi2 known/suspected false positives (numeric IDs, prose)
#   - ordinary coding prompts a developer would actually type
#   - prompts ABOUT credentials but not assigning them
#   - DB_/HOST_/PORT_/VERSION_ patterns that look adjacency-similar
#   - placeholder-form values (<your-key>, ${VAR}, example-*)

#
# Each prompt is scanned in isolation: the loader splits on lines and
# scans each non-blank, non-comment line individually. So adjacency
# across lines does NOT matter for detection — only intra-line content.
