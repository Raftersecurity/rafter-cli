# Calibration: NEGATIVE corpus
#
# Each non-comment, non-blank line is a prompt that MUST produce zero
# detections from prompt-shield. Any detection is a false positive and
# fails calibration.
#
# Comments start with `#`. Blank lines are ignored.
#
# Each section is tagged with a `signal:` class — the underlying defense
# that prevents detection on that class of prompt. A regression in class N
# fails the lines under that section's signal, which makes the failing
# defense visible at a glance. Today's 6 known FPs all live under
# `signal: kebab-non-secret-rhs` (rc-wk5).
#
# Each prompt is scanned in isolation: the loader splits on lines and
# scans each non-blank, non-comment line individually. Cross-line
# adjacency does not matter for detection.
